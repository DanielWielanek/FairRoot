1 MC Step Logger
2 ========
3 
4 Virtual Monte Carlo is calling a Stepping function of FairMCApplication while transporting particles through geometry.
5 MC Step Logger counts individual calls to this Stepping function for each module of the geometry, specified by its name.
6 The logger prints this information after each event, as well as summed up at the end of the run.
7 
8 ## LD_PRELOAD mechanism
9 
10 The library **_preload_** mechanism was used in order to eliminate any negative influence on the time performance when running
11 without the MC Step Logger.
12 
13 The **_preload_** library catches in the runtime all calls to a specificic function (in this case Stepping function of FairMCApplication)
14 and executes a specified list of instructions instead (in this case counting of steps _and_ subsequent execution of the original function).
15 
16 To use the MC Step Logger the user has to **_preload_** the compiled library. It is achieved in one of the following ways:
17 
18 1. Set the **_preload_** library. Run your code. Unset the library:
19 
20 ```bash
21 export LD_PRELOAD=/path/to/your/library/libFairMCStepLogger.so
22 root UserMacro.C
23 unset LD_PRELOAD
24 ```
25 
26 2. Run your code with the **_preload_** library:
27 
28 ```bash
29 LD_PRELOAD=/path/to/your/library/libFairMCStepLogger.so root UserMacro.C
30 ```
31 
32 The PRELOAD mechanism eliminates any overhead when running without the MC Step Logger.
33 It also allowed for absolutely no changes in the FairRoot code.
34 
35 ## MAC OS
36 
37 Since MAC OS has to always be special, then also the **_preload_** looks different here.
38 On MAC OS the user has to specify DYLD_INSERT_LIBRARIES. Yet again, there are two ways to do it:
39 
40 1. Set the **_preload_** library. Run your code. Unset the library:
41 
42 ```bash
43 export DYLD_INSERT_LIBRARIES=/path/to/your/library/libFairMCStepLogger.so
44 root UserMacro.C
45 unset DYLD_INSERT_LIBRARIES
46 ```
47 
48 2. Run your code with the **_preload_** library:
49 
50 ```bash
51 DYLD_INSERT_LIBRARIES=/path/to/your/library/libFairMCStepLogger.so root UserMacro.C
52 ```
