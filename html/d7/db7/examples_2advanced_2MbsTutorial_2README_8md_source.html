4 ## Quick start
5 
6 Enter FairRoot build directory and configure your shell:
7 
8 ~~~~~~~~~~~~~~~~~~
9 . ./config.sh
10 ~~~~~~~~~~~~~~~~~~
11 
12 Enter MbsTutorial macro directory and execute the ROOT macro:
13 
14 ~~~~~~~~~~~~~~~~~~
15 root -l unpack_mbs.C
16 ~~~~~~~~~~~~~~~~~~
17 
18 ## Introduction
19 The MbsTutorial shows how to use MBS data unpacking with FairRunOnline
20 steering class. FairMBSUnpack implements parsing of MBS subevents
21 and creates output in form of array of FairMBSRawItem data objects.
22 This example also demonstrates usage of ROOT histogram online server
23 with possibility to view 1D and 2D histograms, with monitorring option
24 and pre-defined user commands.
25 
26 ## HTTP Server
27 Start of a ROOT Histogram Server can be done by calling the following method
28 of FairRunOnline in the steering macro, before the call to Init()
29 ~~~~~~~~~~~~~~~~~~
30 ActivateHttpServer()
31 ~~~~~~~~~~~~~~~~~~
32 
33 Histograms to be published can be added from an analysis task by:
34 ~~~~~~~~~~~~~~~~~~
35 FairRunOnline::Instance()->AddObject(h1);
36 where h1 is of type TH1F* or TH2F*
37 ~~~~~~~~~~~~~~~~~~
38 
39 In addition, one can publish a canvas by using the same AddObject method. The canvas can be divided into multiple pads.
40 ~~~~~~~~~~~~~~~~~~
41 TCanvas *c1 = new TCanvas("c1", "Title", 10, 10, 500, 500);
42 h1->Draw();
43 FairRunOnline::Instance()->AddObject(c1);
44 ~~~~~~~~~~~~~~~~~~
45 
46 In case of large amount of histogram objects for multiple detectors, the histograms can be organized in folder structure.
47 ~~~~~~~~~~~~~~~~~~
48 TFolder *f1 = new TFolder("myfolder", "Title");
49 f1->Add(h1);
50 FairRunOnline::Instance()->AddObject(f1);
51 ~~~~~~~~~~~~~~~~~~
52 
53 Analysis task can also register a command, which can be executed on
54 a histogram object:
55 ~~~~~~~~~~~~~~~~~~
56 FairRunOnline::Instance()->RegisterHttpCommand("/Reset_h1", "/h1/->Reset()");
57 In this case the command will reset the histogram with name "h1"
58 ~~~~~~~~~~~~~~~~~~
59 
60 During the execution of the macro, histogram server can be accessed
61 by typing "hostname:8080" in any web-browser window on a remote machine,
62 hostname - is the name or IP address of the host running the analysis.
