1 # Move from root to FairMQ continued
2 
3 This example bases on examples/MQ/pixelDetector.
4 
5 The motivation behind is to get rid of TClonesArrays in the data transport.
6 
7 For that, PixelAltFindHits class with ExecMQ() function was created that takes array of PixelPayload::Digi
8 to create an array of PixelPayload::Hit. This eliminates necessity to deal with TClonesArray when
9 reconstructing data.
10 
11 Three devices has been created:
12 - FairMQPixAltSamplerBin: reads data from the binary file and sends PixelPayload::EventHeader and array of PixelPayload::Digi;
13 - FairMQPixAltTaskProcessorBin: runs the PixelAltFindHits task;
14 - FairMQPixAltFileSink: stores the EventHeader and Hits in the output binary file;
15 
16 Preparations:
17 root -l -q 'run_sim.C(100000)' &> sim_100k.dat; root -l -q run_digi.C; root -l -q 'run_digiToBin.C(0)';
18 
19 Script startFairMQPixAlt.sh:
20 - reads the data from binary file (can read root file with "--input-type root" option);
21 - transports and reconstructs binary;
22 - stores root file (can store binary file with "--output-type dat" option);
